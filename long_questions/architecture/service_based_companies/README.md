## 💡 Interview Tips for Service Companies

- **Focus on practical knowledge:** Service companies care about day-to-day Java/Spring Boot experience, not bleeding-edge architecture theory.
- **Project references:** Be ready to explain your past project's architecture — "I worked on a 3-tier Spring Boot application where the backend communicated with a MySQL DB and an Angular frontend."
- **Enterprise context:** Know what tools are common in enterprise IT — Jenkins, SonarQube, Nexus, JBoss/Tomcat, Oracle DB.
- **Architecture diagrams:** Be comfortable drawing a simple architecture diagram on a whiteboard or in Excalidraw.
