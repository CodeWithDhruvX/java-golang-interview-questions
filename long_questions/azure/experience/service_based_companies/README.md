# Azure Interview Questions for Service-Based Companies

This repository contains a curated list of Azure-related interview questions and best practices designed specifically to help you prepare for technical interviews at service-based companies. 

The questions are organized by major topic areas, ensuring comprehensive coverage from fundamental concepts to advanced operations.

## 📂 Content Index

1. [Azure Fundamentals & Core Services](01_azure_fundamentals_and_core_services.md)
   - Covered concepts: Regions, Availability Zones, ARM, IaaS/PaaS/SaaS, Resource Groups, Tags.
2. [Azure Compute Services](02_azure_compute_services.md)
   - Covered concepts: Virtual Machines (VMs), VMSS, App Services, App Service Plans, Azure Functions, Containers vs AKS.
3. [Azure Networking](03_azure_networking.md)
   - Covered concepts: VNets, Subnets, NSG, ASG, Load Balancers, Application Gateway, Traffic Manager, VNet Peering, ExpressRoute.
4. [Azure Storage & Databases](04_azure_storage_and_databases.md)
   - Covered concepts: Blob, File, Table, Queue, Storage Redundancy Options (LRS/GRS), DB Pricing models, SQL Database vs Managed Instance, CosmosDB consistency.
5. [Azure Security & Identity](05_azure_security_and_identity.md)
   - Covered concepts: Microsoft Entra ID (Azure AD), RBAC vs Entra ID Roles, Managed Identities, Key Vault, Conditional Access.
6. [Azure Monitoring & Governance](06_azure_monitoring_and_governance.md)
   - Covered concepts: Azure Monitor, Application Insights, Log Analytics, Azure Policy, Management Groups, Azure Advisor, Budgeting.
7. [Azure Messaging & Integration](07_azure_messaging_and_integration.md)
   - Covered concepts: Azure Logic Apps vs Functions, Service Bus Queues vs Topics, Event Hub vs Event Grid, Dead-Letter Queues, API Management.
8. [Azure DevOps & IaC](08_azure_devops_and_iac.md)
   - Covered concepts: Core DevOps components, CI vs CD, ARM vs Bicep vs Terraform, Service Connections, Pipelines and Release Gates.

## 🎯 Target Audience
These materials are tailored to roles such as:
- Cloud Engineer / Azure Administrator
- System Administrator migrating to the Cloud
- Solutions Architect (Junior / Mid-level)
- DevOps Engineer 

## 💡 How to use this guide
- Use this as a comprehensive study guide before your technical screening.
- Focus on the distinction comparisons (e.g., LB vs App Gateway, Azure Function vs Logic App)—these are frequent questions in interviews.
- Pay attention to the **Best Practices** and **Notes** enclosed within quotes (`>`).
