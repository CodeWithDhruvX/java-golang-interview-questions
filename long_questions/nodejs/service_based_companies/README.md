## 🎯 Interview Focus
For service-based companies, candidates should ensure they have strong foundational knowledge and practical skills. You should comfortably be able to:
- Explain what the Event Loop is.
- Build a basic REST API using Express.
- Understand the difference between `require` and `import`.
- Work with Promises and `async/await`.
- Connect to and query a database via Mongoose or an ORM.
