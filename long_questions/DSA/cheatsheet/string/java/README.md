# Java String Solutions

This folder contains Java implementations for string problems, categorized by topic.

## Files

- [1_String_Basics.java](1_String_Basics.java) - Basic string operations (reverse, palindrome, vowel/consonant count)
- [2_Frequency_Hashing.java](2_Frequency_Hashing.java) - Frequency-based problems (first non-repeating, anagrams, duplicates)
- [3_Substring_Sliding_Window.java](3_Substring_Sliding_Window.java) - Substring problems using sliding window technique
- [4_Palindrome_Problems.java](4_Palindrome_Problems.java) - Advanced palindrome problems
- [5_String_Transformation.java](5_String_Transformation.java) - String transformation problems (reverse words, compression, rotation)
- [6_Pattern_Matching.java](6_Pattern_Matching.java) - Pattern matching and prefix problems
- [7_Group_Anagrams.java](7_Group_Anagrams.java) - Group anagrams problem with multiple approaches

## Usage

Each Java file contains:
- Complete implementations with time and space complexity analysis
- Comprehensive test cases in the main method
- Detailed comments explaining the algorithms

Compile and run any file with:
```bash
javac FileName.java
java FileName
```
