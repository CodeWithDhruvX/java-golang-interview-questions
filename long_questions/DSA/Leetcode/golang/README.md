# 🚀 FAANG LeetCode Preparation - Go Implementation

This repository contains comprehensive Go implementations of all FAANG LeetCode problems organized by the 50 key DSA patterns. Each pattern includes optimized solutions with proper time and space complexity analysis.

## 🚀 How to Use

### Running Individual Problems
Each problem is a self-contained Go program. To run any problem:

```bash
cd path/to/problem/folder
go run problem_name.go
```

For example:
```bash
cd 1_Array_Traversal
go run 485_Max_Consecutive_Ones.go
```

### Building All Programs
To build all programs in a folder:
```bash
cd path/to/folder
go build *.go
```

### Testing
Each program includes comprehensive test cases that demonstrate:
- Basic functionality
- Edge cases
- Performance scenarios
- Input validation
