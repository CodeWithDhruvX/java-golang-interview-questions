# Freelancing Question Bank

Based on the freelancing understanding document, here are the essential questions you should master before starting freelancing.

---

## **Technical Estimation Questions**

### **Project Breakdown & Estimation**
- How do you break down a project into tasks?
- How do you estimate hours for backend vs frontend development?
- What buffer percentage should you add for bugs and client changes?
- How do you calculate total project cost from hourly rates?
- How do you estimate development hours for different components (API, authentication, database, integrations)?
- What's the typical frontend vs backend hour distribution?
- How do you handle combined roles when working alone?

### **Infrastructure & Tool Costing**
- What are typical monthly costs for AWS servers, monitoring, security scanning?
- How do you separate one-time development costs from ongoing monthly costs?
- Who usually pays for infrastructure costs in freelancing projects?
- How do you estimate monitoring and security tool costs?
- What database costs should you factor in?

### **Quick Estimation Techniques**
- How do experienced freelancers estimate projects in 10 minutes?
- What key questions should you ask to estimate quickly (screens, APIs, authentication, integrations, admin dashboard, deployment complexity)?
- How do you price projects between $5k-$50k?
- What milestones should you set for different project types?

## **Essential Formulas & Templates**

### **Core Estimation Formula**
```
Project Price = (Estimated Hours × Hourly Rate) + Risk Buffer (20%) + Tool Costs
```

### **Standard Payment Structure**
- Before starting: 30-50%
- Mid project: 30%
- Final delivery: 20-40%

### **Rate Guidelines by Experience**
- Beginner: $15–$30/hr
- Intermediate: $30–$60/hr
- Experienced: $60–$120/hr

### **Critical Contract Clause**
> "Any additional features or revisions outside the agreed scope will require a separate quote."

## **Quick Reference Checklist**

### **Before Starting Any Project:**
- [ ] Clear Scope of Work document
- [ ] Signed contract with all essential clauses
- [ ] Advance payment received (30-50%)
- [ ] Written communication channels established
- [ ] Milestone structure defined
- [ ] Risk buffer (20%) added to estimates

### **During Project:**
- [ ] All changes documented in writing
- [ ] Scope adherence monitored
- [ ] Milestone payments collected on time
- [ ] Additional features quoted separately

### **Project Completion:**
- [ ] Final payment received before code transfer
- [ ] Ownership transfer completed
- [ ] Project documentation delivered
- [ ] Post-project maintenance terms clarified
