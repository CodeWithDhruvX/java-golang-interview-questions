> Companies like **TCS, Infosys, Wipro, Cognizant, Capgemini, Accenture, HCL, Tech Mahindra**, etc.

Service-based companies focus on **practical JavaScript skills**, DOM manipulation, REST API integration, ES6+ features, and React basics. Interviews are **moderate in depth** and emphasize real-world web development skills, problem solving, and project experience.

## 🎯 Interview Focus Areas (Service Companies)

- ✅ **JavaScript Fundamentals**: var/let/const, hoisting, closures, `this`, type coercion
- ✅ **DOM**: selecting elements, event listeners, form validation, event delegation
- ✅ **ES6+**: arrow functions, destructuring, template literals, spread/rest, classes
- ✅ **Async**: event loop, Promises, async/await error handling, fetch API
- ✅ **OOP**: prototype, class syntax, inheritance, design patterns (Singleton, Factory)
- ✅ **Error Handling**: try/catch, custom errors, async error handling
- ✅ **Node.js Basics**: require/import, Express API, npm, environment variables
- ✅ **Security**: XSS prevention, localStorage vs cookies, CORS basics

---

## 💡 Tips for Service Company JavaScript Interviews

1. **Master the fundamentals** — closures, hoisting, `this` binding, prototype chain
2. **Know ES6+ well** — destructuring, arrow functions, spread, async/await
3. **DOM manipulation** — event delegation, debounce, form validation are common
4. **Async patterns** — be able to explain and write Promises and async/await
5. **fetch API** — know how to make GET/POST requests, handle errors
6. **localStorage basics** — when to use it, what not to store there (tokens!)
7. **Code quality** — clean code principles, pure functions, meaningful names
8. **Node.js basics** — npm, package.json, Express routes are often asked
