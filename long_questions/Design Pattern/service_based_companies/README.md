# Design Pattern Interview Questions - Service Based Companies

This section contains frequently asked Design Pattern interview questions in service-based companies (e.g., TCS, Infosys, Wipro, Cognizant, IBM, Tech Mahindra, etc.).

The questions are typically focused on the basic understanding of design patterns, their types, and straightforward implementations in Java or other Object-Oriented languages.

## Topics Covered:
1. **[Creational Patterns](./01_creational_patterns.md)**
2. **[Structural Patterns](./02_structural_patterns.md)**
3. **[Behavioral Patterns](./03_behavioral_patterns.md)**

## Focus Areas for Service-Based Companies
*   **Definitions:** Clear and concise definitions of common patterns (Singleton, Factory, Observer).
*   **Categories:** The differences between Creational, Structural, and Behavioral patterns.
*   **Real-time Examples:** Ability to map a design pattern to a simple real-world scenario.
*   **Code Snippets:** Writing a basic implementation of patterns like Singleton (with double-checked locking) or Factory.
*   **Trade-offs:** Basic advantages and disadvantages of using a specific pattern.
*   **OOP Concepts:** How design patterns relate to core OOP concepts (Encapsulation, Inheritance, Polymorphism, Abstraction) and SOLID principles.
