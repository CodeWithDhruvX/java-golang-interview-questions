## 💡 How to Use

1. **Read** each question and try to answer before looking at the explanation.
2. **Focus on the "why"** — don't just memorize outputs, understand the mechanism.
3. Use the **reading progress** tracker inside each file (`🔖 Last read:`).
4. For concurrency, run the code to observe actual behavior.
