# Java Version Examples

This directory contains executable Java programs demonstrating the key features introduced in major Java versions.

## 📂 File List

1.  **[Java1_0_HelloWorld.java](./Java1_0_HelloWorld.java)**: The beginning - WORA and JVM.
2.  **[Java1_1_InnerClasses.java](./Java1_1_InnerClasses.java)**: Inner classes, JDBC, RMI.
3.  **[Java1_2_Collections.java](./Java1_2_Collections.java)**: The Collections Framework (List, Set, Map).
4.  **[Java1_3_Features.java](./Java1_3_Features.java)**: HotSpot JVM and Java Sound.
5.  **[Java1_4_Features.java](./Java1_4_Features.java)**: Assertions, Logging API, and NIO.
6.  **[Java5_Features.java](./Java5_Features.java)**: Generics, Enums, Autoboxing, and Varargs.
7.  **[Java6_Features.java](./Java6_Features.java)**: Scripting support, JDBC 4.0, and Console API.
8.  **[Java7_Features.java](./Java7_Features.java)**: Try-with-resources, Diamond operator, and String in switch.
9.  **[Java8_Features.java](./Java8_Features.java)**: Lambdas, Streams, Optional, and the new Date/Time API.
10. **[Java9_Features.java](./Java9_Features.java)**: Factory methods for collections and `takeWhile`.
11. **[Java10_Features.java](./Java10_Features.java)**: Local variable type inference (`var`).
12. **[Java11_Features.java](./Java11_Features.java)**: New String methods and standard HTTP Client.
13. **[Java12_13_Features.java](./Java12_13_Features.java)**: Switch expressions and Text Blocks (Previews).
14. **[Java14_Features.java](./Java14_Features.java)**: Records and Pattern Matching for `instanceof`.
15. **[Java15_16_Features.java](./Java15_16_Features.java)**: Sealed classes (preview) and standardized Records.
16. **[Java17_Features.java](./Java17_Features.java)**: Sealed Classes and standard Pattern Matching.
17. **[Java18_WebServer.java](./Java18_WebServer.java)**: Simple Web Server and UTF-8 by default.
18. **[Java19_20_Features.java](./Java19_20_Features.java)**: Virtual Threads and Record Patterns (Previews).
19. **[Java21_Features.java](./Java21_Features.java)**: Virtual Threads and Sequenced Collections.
20. **[Java_Latest_Features.java](./Java_Latest_Features.java)**: Overview of Java 22/23 updates.

## 🚀 How to Run

To run these files, ensure you have the required JDK version installed.

```bash
# Example: Running Java 8 features (if you have JDK 8+)
javac Java8_Features.java
java Java8_Features
```

> **Note:** Files like `Java21_Features.java` require JDK 21 or higher to compile and run.
