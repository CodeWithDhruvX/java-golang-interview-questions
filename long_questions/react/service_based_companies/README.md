# 📘 React Interview Questions — Service-Based Companies

> Companies like **TCS, Infosys, Wipro, Cognizant, HCL, Capgemini, Accenture, Tech Mahindra, Mphasis**, etc.

Service-based companies focus on **fundamental React knowledge**, practical component building, REST API integration, and basic state management. Questions are straightforward and test if you can build and maintain real-world React applications.

## 🎯 Interview Focus Areas (Service Companies)

- ✅ React fundamentals: JSX, virtual DOM, rendering
- ✅ Component patterns: functional, class, lifecycle
- ✅ Hooks: useState, useEffect, useRef, useContext
- ✅ Forms: controlled/uncontrolled, validation, submission
- ✅ API integration: fetch, axios, error and loading handling
- ✅ Routing: React Router DOM (v6), nested routes, params
- ✅ Styling: CSS Modules, Styled Components, popular UI libraries
- ✅ Deployment: build tools, environment variables, hosting

---

## 💡 Tips for Service Company Interviews

1. **Master the basics** — JSX, props, state, lifecycle methods
2. **Know all common hooks** — especially useEffect dependency array
3. **Practice forms** — controlled inputs, validation, submit handlers
4. **REST API integration** — loading, error, success state patterns
5. **React Router v6** — routes, Link, useNavigate, useParams
6. **UI libraries** — Material UI or Ant Design component usage
7. **Basic testing** — RTL render + userEvent + getByRole queries
