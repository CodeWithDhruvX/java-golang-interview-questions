# Microfrontend Interview Questions Directory

This directory contains comprehensive interview questions and answers for Microfrontend architectures. 

The questions are organized to target different tiers of technical interviews:

## Structure

*   **/product_based_companies**: Deep, architectural, and systemic questions expected in top-tier or product-based companies. Evaluates trade-offs, advanced Webpack Module Federation knowledge, and high-scale CI/CD deployment strategies.
*   **/service_based_companies**: Fundamental understanding, framework knowledge (Single-SPA, Module Federation basics), and core concepts typically asked in consultancy or service-based IT organizations.

## Topics Covered

*   Advanced Architecture & Monolith Replacement Strategies
*   Webpack Module Federation In-Depth
*   State Management & Cross-MFE Communication (Pub/Sub)
*   Testing (Contract/E2E) & Zero-Downtime Deployments
*   Single-SPA Routing & Component Lifecycles
*   CSS Isolation (Shadow DOM, CSS Modules)
