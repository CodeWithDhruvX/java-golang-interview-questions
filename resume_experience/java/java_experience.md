## **The Strategic Lead: Ascendion Pvt. Ltd.**

### **Insight Governance (KPMG)**

"In my most recent role at Ascendion, I was tasked with a high-stakes challenge for KPMG involving their Insight Governance project. The client needed a way to turn massive datasets into actionable insights, so I spearheaded the integration of PowerBI dashboards directly into our UI. To ensure the platform could handle the scale, I architected a microservices ecosystem using Docker and Kubernetes on AKS. One of the most complex parts was handling high-throughput file uploads, for which I designed an asynchronous pipeline using Azure Blob Storage. By the time we integrated full CI/CD via Azure DevOps with automated quality gates like SonarLint, we saw a 15% boost in operational efficiency and a much faster delivery cycle."

### **KPMG Audit Platform**

"Before that, I led the backend architecture for the KPMG Audit Platform. This was a lesson in resilience; we needed to support multiple concurrent audit processes without any lag or security risks. I used Java Spring Boot and Spring Cloud to build a scalable foundation and implemented very strict role-based access controls using Spring Security. A major win for the team was when I developed a library of reusable service components for our calculations, which actually ended up reducing our development time for new features by about 25%. It was all about making the system secure yet flexible."

## **What "Perfect Spoken Format" Looks Like**

When you speak this in an interview, remember that you are not reading a list; you are telling a story of **growth**.

* 
**The Transition:** Start by saying, *"Throughout my 6 years, I’ve moved from focusing on clean code to focusing on cloud-native architecture and business efficiency"*.


* **The Tone:** Be confident but collaborative. Mention how you *"collaborated with QA teams"* or *"mentored junior developers"*.


* **The Metric:** Always end a project story with the result. Instead of saying "I used Kubernetes," say, *"I used Kubernetes, which allowed us to achieve high availability for our critical services"*.


* 
**The Passion:** Mention your GitHub project (JobPortal) as a way to show you are "always learning" best practices in your own time.
